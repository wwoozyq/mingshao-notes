# 模式识别与机器学习

这页可以作为你以后补机器学习笔记的总入口，先把框架搭出来，后面逐步往里填。

## 可以记录的模块

- 监督学习：分类、回归、损失函数、评估指标
- 无监督学习：聚类、降维、特征表示
- 常见模型：线性回归、逻辑回归、SVM、决策树、随机森林、KNN
- 深度学习基础：神经网络、反向传播、优化器、过拟合

## 推荐写法

- 先写概念，再写公式和例题
- 每个模型都补“适用场景”和“优缺点”
- 记录你最容易混淆的点，比如 bias / variance、precision / recall
