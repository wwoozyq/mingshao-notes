# 自然语言处理

这页可以作为 NLP 方向的入口，后面你可以把课程笔记、论文理解和项目实战都挂到这里。

## 可以记录的主题

- 文本预处理：分词、清洗、停用词、词向量
- 经典方法：TF-IDF、朴素贝叶斯、RNN、LSTM
- 现代方法：Transformer、BERT、大语言模型
- 任务方向：文本分类、命名实体识别、机器翻译、问答、摘要

## 推荐写法

- 写清楚每个方法解决什么问题
- 把模型结构图和关键概念用自己的话讲一遍
- 补充你自己做过的例子或实验结果
