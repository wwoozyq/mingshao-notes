# AI 总览

这个栏目适合记录你对 AI 的长期学习和实践，不只是“收藏链接”，而是把自己的理解写出来。

## 可以分成哪些方向

- 学习路线：阶段目标、课程、论文、工具和项目安排
- 模式识别与机器学习：分类、回归、聚类、模型评估、常见算法
- 自然语言处理：文本表示、语言模型、序列任务、应用实践
- 项目实践：你自己做过的 demo、网站、脚本、实验

## 推荐页面类型

- 一篇技术主题笔记
- 一个工具实测报告
- 一次项目复盘
- 一份学习路线图
