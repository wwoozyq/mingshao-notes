# AI 学习路线

这页适合当作你的 AI 学习总入口，后面可以持续往里补链接和阶段目标。

## 第一阶段：建立基本概念

- 什么是大语言模型
- Prompt、上下文、Token、Embedding 分别是什么
- 常见产品和 API 有什么区别
- 机器学习、深度学习、自然语言处理之间的关系是什么

## 第二阶段：开始动手

- 练习把真实问题拆成清晰提示词
- 尝试调用一次模型 API
- 做一个很小的自动化项目，比如问答、摘要或分类
- 补一页你自己的机器学习基础概念笔记

## 第三阶段：形成自己的方法

- 记录哪些提示词结构对你最有效
- 对比不同模型在同一任务上的结果
- 把项目过程、踩坑和判断写成笔记
