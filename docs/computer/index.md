# 计算机总览

这个栏目现在聚焦你最明确的两块内容：数据结构基础，以及 LeetCode Hot100 刷题记录。

## 适合记录的内容

- 数据结构基础：数组、链表、栈、队列、树、图、哈希
- Hot100 刷题：题目链接、思路、代码、复盘
- 常见算法模板：双指针、滑动窗口、二分、DFS、BFS、动态规划
- 易错点总结：边界条件、复杂度、常见 bug

## 推荐写法

- 一篇笔记只回答一个明确问题
- 每道题都留下自己的思路，不只贴答案
- 把容易忘的模板和坑点单独整理出来

## 起步建议

你可以先从 [数据结构基础](cs-fundamentals.md) 和 [Hot100 刷题](hot100.md) 这两页开始。
