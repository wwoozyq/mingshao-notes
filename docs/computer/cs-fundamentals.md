# 数据结构基础

这页适合做你数据结构笔记的总入口，以后可以把每个主题拆成更详细的页面。

## 可以逐步展开的主题

- 数据结构：数组、链表、栈、队列、树、图、哈希
- 常见操作：插入、删除、查找、遍历、反转、排序
- 高频思路：双指针、滑动窗口、递归、回溯、堆、并查集
- 刷题连接：这些基础知识最终都可以映射到 Hot100 题单里

## 建议的写法

每个主题都可以按这个顺序写：

1. 它解决什么问题
2. 核心概念是什么
3. 典型题目会怎么考
4. 你自己的例子或代码
