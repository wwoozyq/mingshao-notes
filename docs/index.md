## 使用方式

- 在 `docs/` 里新建 Markdown 文件
- 在 `mkdocs.yml` 里加到导航
- 运行 `mkdocs serve` 预览

## 常用命令

```bash
source .venv/bin/activate
mkdocs serve
mkdocs build
```
